How to use this dataset¶

Unless you have a very specialist requirement, don’t download this dataset just to get the observations, get them from ICOADS release 3 instead.

Do download this dataset to add more expeditions - there is a ToDo section with some that are ready for conversion.

This document is kept under version control in a git repository. The repository is hosted on GitHub (and the documentation made with GitHub Pages). The repository is https://github.com/oldweather/Expeditions.

If you are familiar with GitHub, you already know what to do: If you’d prefer not to bother with that, you can download the whole dataset as a zip file.

The scripts in this package depend on the IMMA toolkit which provides both support for the IMMA data format and an implementation of the lmrlib data conversion functions.
